Générateur d'icônes PWA

Instructions :

  1. Cliquez sur le bouton "Générer les icônes" ci-dessous
  2. Les icônes s'afficheront avec un aperçu
  3. Faites un clic droit sur chaque icône et sélectionnez "Enregistrer l'image sous..."
  4. Enregistrez chaque icône dans le dossier public/assets/icons/ avec le nom correspondant à sa taille (ex: icon-192x192.png)
